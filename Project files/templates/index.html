DIABETIC RETINOPATHY

ABOUT PROJECT

Problem

Diabetic Retinopathy is a serious complication of diabetes that damages the retina and can lead to blindness. Early detection is difficult and manual diagnosis is time-consuming.

Solution

This project uses deep learning techniques to automatically detect and classify Diabetic Retinopathy into multiple stages from retinal images, assisting doctors in faster diagnosis.
